- [WireGuard](https://wireguard.com/install/): WireGuard is a fast, modern, and secure VPN tunnel. + [WireGuard Config by ProtonVPN](https://account.protonvpn.com/downloads#wireguard-configuration): The best VPN for speed and security.
  <details><summary>
  WG Setup
  </summary>
 
  - Install
   ```
   sudo dnf install wireguard-tools
   ```
  - Generate a private & public key pair for Fedora workstation
   ```
   wg genkey | tee privatekey | wg pubkey | sudo tee /etc/wireguard/publickey
   ```
  - Copy downloaded proton-vpn wg config to `/etc/wireguard/`
   ```
   sudo nmcli con import type wireguard file /etc/wireguard/wg-US-FREE-2.conf
   ```
  - Import wg config into Fedora’s NetworkManager (Fedora's `Settings` > `Network` > `VPN`)
   ```
   sudo nmcli con import type wireguard file /etc/wireguard/wg-US-FREE-2.conf
   ```
  - CLI
  ```
  sudo nmcli connection show
  ```
  ```
  sudo wg-quick up wg-US-FREE-2
  ```
  ```
  sudo wg show
  ```
  ```
  sudo wg-quick down wg-US-FREE-2
  ```
  - Uninstall
  ```
  sudo nmcli connection show
  ```
  ```
  sudo nmcli connection delete wg-US-FREE-2.conf
  ```
  ```
  sudo dnf remove wireguard-tools
  ```
  </details>
